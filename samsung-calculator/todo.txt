- Botão "0":
  - Não deveria ser possível digitar vários zeros quando o número é iniciado ou composto apenas por zeros, por exemplo, [000001, 00000].
  - Apenas deveria ser possível quando o número é iniciado por algum outro número diferente de zero.

- Botão "()":
  - Pode funcionar com o display vazio.
  - Não deve funcionar caso o último caractere for uma operação.
  - Os parenteses não devem ser fechados caso não haja números entre eles.
  - O evaluator deve identificar e lançar um erro caso haja um parenteses não fechado.

- Botão "+/-":
  - Altera o sinal de um número.
  - Só deve funcionar quando o display estiver vazio ou quando uma operação for o último caractere.
  - Caso o display esteja vazio apenas adiciona "-" ao display.
  - Adiciona ou remove sinal negativo ao número.

- Botão ",":
  - Não deve fazer nada caso o display esteja vazio.
  - Não deve fazer nada caso o último caractere seja uma operação.
  - Não deve fazer nada caso o último caractere seja um parenteses.
  - Não deve funcionar caso o número já possua vírgula.
